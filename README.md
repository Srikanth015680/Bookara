# Bookara 📚

A full-stack book store application.

##  Structure
- Admin Panel (React + Vite)
- User Frontend (React + Vite)
- Backend (Node.js + Express)

##  Features
- User Authentication
- Product Management
- Cart & Orders
- Admin Dashboard

##  Tech Stack
- React
- Node.js
- Express
- MongoDB

##  Project Setup
cd frontend
npm install
npm run dev
